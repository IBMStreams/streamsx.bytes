com.ibm.streamsx.bytes::native.function.function

The page documents the properties of a set of native functions that may be invoked from SPL. The properties are used to map the SPL function interface to the native C++ code and libraries that implement the function. It's assumed that the function prototypes for the native code are defined in a C++ header file. Each header file corresponds to Function Set, which declares the header file, a CPP namespace, and the set of SPL interfaces for each of the native functions. If native functions require libraries, the library dependencies are also specified in the function set.
